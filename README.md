# Tutor LMS Plugins

Custom WordPress plugins built to extend Tutor LMS.

Each plugin lives in its own subfolder with a dedicated README.

## Plugins

| Folder | Description |
|--------|-------------|
| [tutor-lms-course-info-shortcodes](./tutor-lms-course-info-shortcodes/) | Shortcodes for course summary, instructors, duration, last update, and enrollment count |

## Requirements

- WordPress 5.0+
- Tutor LMS plugin
