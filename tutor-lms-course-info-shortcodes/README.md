# Tutor LMS Course Info Shortcodes

Provides shortcodes to display Tutor LMS course meta information inside loop templates or anywhere on the page.

## Shortcodes

| Shortcode | Description |
|-----------|-------------|
| `[course_summary]` | Displays the course excerpt/summary |
| `[course_instructors]` | Displays instructor photos and names |
| `[course_last_update]` | Displays the last updated date |
| `[course_duration]` | Displays the total course duration |
| `[course_enrollment]` | Displays total enrolled student count |

## Usage

1. Paste `tutor-lms-course-info-shortcodes.php` into your theme's `functions.php` or a custom plugin.
2. Use the shortcodes inside Elementor, Gutenberg, or any template.

## Requirements

- WordPress 5.0+
- Tutor LMS plugin
